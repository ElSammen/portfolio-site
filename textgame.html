Projects

The objective of this project was to better understand Object Oriented Programming by making a text based adventure game centered on a single HTML page.

In order to do this I first created the initial HTML, I then created a few JavaScript classes to function as rooms, characters and items respectively. After, using the previously established setter feature(assigned at class creation for assigning attributes) I fleshed out the Room class with its name, a description, the characters / items inside and rooms linked to it (and the direction). I then created a function for later use that checked which rooms were linked.

I then created content for the item and character classes, linked those to rooms and created a function to provide the name/description of the room you're in, the direction of any linked rooms, any characters and or items you may find (This function would then replace the relevant html info with the provided descriptions). After which, I then created a function for movement, which would accept inputs of north/east/south/west and provided it matched the direction of a linked room, would allow you to move to that room, providing you with a new room description etc.
