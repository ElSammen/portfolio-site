JavaScript Quiz


GitHub Repo

The objective of this project was to create a quiz using largely JavaScript, in order to better understand form validation & JavaScript itself.

In order to do this I first created the basic html for the quiz to take place in, featuring 4 buttons for answer selection. After tailoring that I created an object array of questions and answers, with 3 flagged as wrong answers and 1 correct. Then I created a function that would populate the html document with the questions and answers (Using Math.random to select a handful of the questions from a larger list for replayability).

After, I created another function that would detect whether the answer was correct or not using form validation, adding to the users running score in the background. After the user had answered all available questions, the function would than replace the main quiz box with the users score and provide them with the option to replay.
